# Portfolio slideshow images

Put exactly 5 screenshots per website (optimized JPG/WebP recommended, ~720–900px wide):

  portfolio-preview/slides/{slug}/01.jpg
  portfolio-preview/slides/{slug}/02.jpg
  ...
  portfolio-preview/slides/{slug}/05.jpg

Optional full-page hover scroll source (tall screenshot):

  portfolio-preview/slides/{slug}/full.jpg

Slugs: ghorsajan, real-sign-bd, gully-apparel, gozero-print, zaiax

Tips:
- Prefer JPG quality ~65–75 or WebP for fast loading
- Full-page shots should be resized (max width ~800px) before upload
- Do not use multi-MB raw captures — they cause lag
